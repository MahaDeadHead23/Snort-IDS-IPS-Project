1.Before doing anything, it's recommended to update making sure everything will be working correctly

sudo apt-get update sudo apt-get upgrade

2.Install Dependecies

sudo apt-get install build-essential libpcap-dev libpcre3-dev libdnet-dev zlib1g-dev libluajit-5.1-dev openssl libssl-dev sudo apt-get install libdaq-dev libdaq2 libhwloc-dev cmake

3. Install Snort

sudo apt-get install snort

4. Check Snort Installation

snort -V or snort --version

5.Configure Snort sudo nano /etc/snort/snort.conf

YOU MAY NEED TO SET UP YOUR NETWORK VARIABLES :

var HOME_NET [your_network_ip_range] - Refer to the Screenshots

var EXTERNAL_NET any

6. Test Snort Configuration sudo snort -T -c /etc/snort/snort.conf

7. Running Snort in Packet Logging Mode

sudo snort -dev -l /var/log/snort -i eth0

8.Running Snort in IDS Mode

sudo snort -A console -q -c /etc/snort/snort.conf -i eth0

9. Installing DAQ for IPS Mode

sudo apt-get install daq

10. Set Up IP Tables for IPS Mode

sudo iptables -A INPUT -j NFQUEUE --queue-num 0 sudo iptables -A FORWARD -j NFQUEUE --queue-num 0

11. Running Snort in IPS Mode

sudo snort -A console -Q --daq nfq --daq-var queue=0 -c /etc/snort/snort.conf -i eth0

12. STOP Snort

Cnrl + C
